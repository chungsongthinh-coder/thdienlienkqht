⚙️ Cài đặt kết nối Google Sheets

📋 Hướng dẫn nhanh
  1. Tạo Google Sheet mới → copy Spreadsheet ID từ URL
  2. Mở Extensions → Apps Script → dán nội dung file Code.gs
  3. Đổi SPREADSHEET_ID trong Code.gs → Chạy setupSheets()
  4. Deploy → New deployment → Web app → Execute as: Me, Access: Anyone
  5. Copy Web App URL → dán vào ô dưới đây

URL sẽ được lưu trong trình duyệt. Mỗi máy tính cần nhập lại một lần.

👤 Tài khoản đăng nhập (quản lý trong Google Sheets → sheet Users)

Để thêm/sửa tài khoản, mở Google Sheets → sheet Users:
Các cột: username | password | hoten | role | lop_phu_trach
Role: admin (xem tất cả) hoặc teacher (chỉ xem lớp mình phụ trách)
