⚙️ Cài đặt kết nối Google Sheets

📋 Hướng dẫn lấy Web App URL
  1. Google Sheet → Extensions → Apps Script → dán Code.gs mới → Lưu
  2. Click Triển khai → Quản lý triển khai → ✏️ → Phiên bản mới → Triển khai
  3. Copy URL dạng: .../macros/s/AKfycb.../exec
⚠️ Bắt buộc: Execute as = Me | Who has access = Anyone

Mỗi lần sửa Code.gs phải Deploy lại phiên bản mới!

URL dạng .../exec — KHÔNG phải URL có chữ /edit hoặc /dev
