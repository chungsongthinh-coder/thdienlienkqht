⚙️ Cài đặt kết nối Google Sheets

📋 Hướng dẫn lấy Web App URL
  1. Mở Google Sheet → Extensions → Apps Script
  2. Nhấn Deploy → Manage deployments → ✏️
  3. Chọn Version: New version → Deploy
  4. Copy URL dạng: ...macros/s/AKfycb.../exec
⚠️ Quan trọng khi Deploy

• Execute as: Me  |  Who has access: Anyone (không phải "Anyone with Google account")
• Mỗi lần sửa Code.gs phải Deploy lại version mới!
